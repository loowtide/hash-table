# hash_table.h

---

A simple header file implementing hash tables.

## Usage

---

Just `#include "hash_table.h"` in your code!

The current supported platforms are Linux, macOS and Windows.

The current supported compilers are gcc, clang(These are the ones i checked).

## Building files

---

`gcc <file-name> hash_table.c -lm` \
or \
`clang <file-name> hash__table.c -lm`
